# Cabinet photos

Drop minister portrait files in this directory as `<slug>.jpg` (or `.webp`). The Avatar component on `/cabinet` and `/cabinet/<slug>` picks them up automatically when the matching MDX frontmatter sets `photo: /cabinet/<slug>.jpg`.

See `CONTRIBUTING.md § Cabinet photos` for sourcing rules, licence requirements, and file specs. AI-generated portraits, news stock photos, and social-media profile pictures are not admissible.

When you add a file, record the source URL and licence in `attribution.txt` in this directory so the audit trail is honest.
