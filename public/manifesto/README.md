# Manifesto archive

The UDF 2026 election manifesto PDF lives in this directory as `udf-2026.pdf`.

The file is not yet committed. The maintainer drops the canonical PDF here, sourced from the Indian National Congress release of 2 April 2026, after which `https://keralapromises.in/manifesto/udf-2026.pdf` resolves to it.

The placeholder promises in `src/content/promises/` cite this URL today; once the PDF lands they cite a stable, hosted copy, and the archive.org mirror Action records a permanent snapshot on first publish.
